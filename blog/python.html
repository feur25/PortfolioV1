Python Qu'es que c'est ??

Bonne question, eh bien tout d'abord c'est un language de programmation vous n'êtes pas sans savoir, que c'est une notation conventionnelle destinée à formuler des algorithmes et produire des programmes informatiques qui les appliquent. Rien avoir donc avec le Python(serpent). Grâce à cet outil incroyable et je pèse mes mots, vous pourrez créé des jeux si l'envie vous en prend.

Python est donc un langage de programmation interprété, et multiplateformes. Orientée objet. Juste au dessus vous pouvait apercevoir une vidéo, qui traite sur 10 épisodes la création d'un jeux vidéo en python. Mais avant tout cela il faudrait d'abord savoir comment installer python, voici les étapes a suivres pour pouvoir commencer a travailler avec python :

Rendez-vous sur le site Officel : Lien vers Le site Officiel de Python

Télécharger la version souhaitez, puis lancer le setup, pour ensuite installer tous les prérequis de base nécessaire au lancement de l'IDLE de python, c'est dernier ne comporte pas les modules externes, comme numpy, pygame, etc .qui devront être importé via l'invite de commande par exemple (cmd)
